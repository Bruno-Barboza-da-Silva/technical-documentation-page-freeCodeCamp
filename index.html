Using Specific Tags

After make the basic tags, it's necessary fill this thags whit some content, and for that there is a lot of tags, but the most used ones are: <p>, <h1> ... <h6> <img>, <iframe>, <form> <audio>, <a>. It's important to say that some of that tags can contein other tags.

  • <p> This is a tag for the developer insert the content in text format. must contain at most one paragraph;
  • <h1> ... <h6> This is a tag for a developer insert a title in text format. The <h1> is the tag that has the most prominence and the <h6> is the tag that has less prominence;
  • <img> This tag is for inserting an image;
  • <iframe>This tag is to insert content (image, audio or video) present on another page;
  • <form>This tag is for inserting a form on the page. This one needs other tags inside it like the input and label tags;
  • <audio> This tag is for insert a audio in the page;
  • <a>This tag makes the text bold.
